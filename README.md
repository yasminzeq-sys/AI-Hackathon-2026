# Smart E-Commerce: Real-Time Hesitation Detector 🛒🤖

### Project Overview
Our AI predicts when a customer is about to leave a site without buying and triggers a "Social Proof" to increase sales.

### Tech Stack
* **Frontend:** Streamlit
* **AI Model:** Random Forest Classifier (Scikit-Learn)
* **Logic:** Customer Psychology (Helping them decide)

### How to Run
1. Adjust the sliders (Time on Site / Pages Viewed).
2. Observe the AI Status and the dynamic 'Popularity Alert' output.
